git init.               -> Cria um repo git na pasta atual 
git status              -> Mostra o status do trabalho (da cada arquivo)
git add nome_arquivo    -> Adiciona arquivo para ser commitado
git commit -m "msg"     -> Faz commit do que estava para ser commitado
git log                 -> Mostra todo o hitorico de commits



## FLUXO DE TRABALHO
1.criar o diretório (pasta) do projeto
2.iniciar o git dentro diretório criado (git init .)
3.editar ou criar arquivos
4.git status
5.git add <nome arquivo>
6.git status
7.git commit -m "Mensagem estranha"
8.git status
9.volta para o passo 3
